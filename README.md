# ORM-BOOK-API-TEST

## Deskripsi Project
Its a unit test for `book service` task. it's used a `testify` and `go mock` library to test it. there's 4 endpoint that would be test:
- Get All Books
- Get Book by ID
- Create Book
- Update Book
- Delete Book
